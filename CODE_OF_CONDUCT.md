# Code of Conduct

## Our Pledge

We want participation in the `comon_opentelemetry` project to be a harassment-free
experience for everyone, regardless of age, body size, disability, ethnicity,
gender identity and expression, experience level, education, socio-economic
status, nationality, personal appearance, race, religion, or sexual identity
and orientation.

## Expected Behavior

- Be respectful in discussions and code review.
- Assume good intent, but be precise when discussing technical disagreements.
- Focus feedback on code, behavior, and decisions rather than people.
- Help keep issue and PR discussions on-topic and actionable.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language
- Personal attacks or sustained hostility
- Publishing private information without explicit permission
- Trolling, intimidation, or deliberately derailing discussions

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code,
issues, and other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a private channel with
the maintainers through the repository contact options. Reports will be
reviewed promptly and handled with discretion.

## Scope

This Code of Conduct applies to repository discussions, issue trackers, pull
requests, and any project-managed communication spaces.
